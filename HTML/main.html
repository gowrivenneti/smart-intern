Welcome to CSS

There are three ways to add css in html file

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS

We can add css by using style attribute within in the html tags

Internal CSS

We can write css inside the style tag. It is only used for single Page

External CSS

Styles are written in external .css file and is linked to html file. It is reusable

CSS SELECTORS

  1. Universal (*)
  2. Element (tags)
  3. Class (.)
  4. ID (#)
